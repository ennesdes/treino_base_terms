# treino_base_terms

Site estático do **Treino Base** (GitHub Pages) — página inicial, termos de uso e política de privacidade em URLs separadas para conformidade com Google Cloud OAuth.

## URLs publicadas

| Página | URL |
|--------|-----|
| Página inicial do app | `https://ennesdes.github.io/treino_base_terms/` |
| Termos de Uso | `https://ennesdes.github.io/treino_base_terms/termos.html` |
| Política de Privacidade | `https://ennesdes.github.io/treino_base_terms/privacidade.html` |

## Google Cloud Console (OAuth)

Na tela de consentimento OAuth, use:

- **Página inicial do aplicativo:** `https://ennesdes.github.io/treino_base_terms/`
- **Link da Política de Privacidade:** `https://ennesdes.github.io/treino_base_terms/privacidade.html`
- **Link dos Termos de Serviço:** `https://ennesdes.github.io/treino_base_terms/termos.html`
- **Domínio autorizado:** `ennesdes.github.io`

## Deploy

Faça push para o branch `main` do repositório GitHub com GitHub Pages habilitado (source: branch main, pasta `/`).
